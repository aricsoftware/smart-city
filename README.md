# ATL Smart City — Command Center

A real-time smart city dashboard for the City of Atlanta, featuring interactive mapping, vehicle tracking, environmental monitoring, and GIS platform integration.

## Features

- **Live Interactive Map** — Leaflet.js + OpenStreetMap with CartoDB Dark tiles
- **Vehicle Tracking** — Police cars, ambulances, and school buses moving in closed-loop routes
- **Traffic Signal Monitoring** — 10 traffic lights with real-time status
- **GIS Platform** — Heatmap overlays, neighborhood zones, multi-layer controls
- **Dashboard Modules** — Public safety, traffic, environment, energy, water, waste, transit, 311
- **Real-time Simulation** — KPIs, alerts, and environmental sensors update live

## Tech Stack

- Tailwind CSS v4 (CDN)
- Leaflet.js + OpenStreetMap
- Chart.js
- Font Awesome 6
- Vanilla JavaScript (no build step)
